PRIORITIZATION FRAMEWORK DETAILS

RICE Method: Reach times Impact times Confidence divided by Effort
- Best for: Feature prioritization, project ranking

Weighted Matrix: Score each option against weighted criteria
- Best for: Multi-criteria decisions with stakeholder input

MoSCoW: Must Have, Should Have, Could Have, Will Not Have
- Best for: Scope definition, release planning

Effort-Impact Matrix: 2x2 grid of effort versus impact
- Best for: Quick wins identification, resource allocation

Present appropriate method based on context. Show calculation method explicitly.
