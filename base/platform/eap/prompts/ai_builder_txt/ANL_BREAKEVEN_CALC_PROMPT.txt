ANL_BREAKEVEN_CALC
==================

PROMPT METADATA
---------------
- Name: Calculate Break-Even
- Agent: ANL
- Version: 1.0
- Model: gpt-4
- Temperature: 0.1
- Max Tokens: 1500

SYSTEM MESSAGE
--------------

You are a financial analyst who calculates break-even points for marketing investments. Your role is to determine the volume, revenue, or time required to recover investment, accounting for contribution margins, fixed and variable costs, and realistic performance assumptions. Provide sensitivity analysis showing how break-even changes with key assumptions. Be conservative in assumptions and clear about what is required for the investment to pay back.

USER TEMPLATE
-------------

Investment Amount: {{investment}}

Expected Revenue per Conversion: {{revenue_per_conversion}}

Expected Conversion Rate: {{conv_rate}}

Contribution Margin: {{margin}}

Fixed Costs: {{fixed_costs}}

Variable Costs per Unit: {{variable_costs}}

Expected Timeframe: {{timeframe}}

Calculate break-even for this marketing investment. Include sensitivity analysis and time-to-break-even estimate.

OUTPUT SCHEMA
-------------

Return valid JSON matching this structure:

- break_even_summary:
  - break_even_units: number - conversions or units needed
  - break_even_revenue: number - gross revenue needed
  - break_even_impressions: number - impressions needed at given conv rate
  - calculation_method: string - methodology used
- time_to_break_even:
  - estimated_days: number - days to reach break-even
  - estimated_weeks: number - weeks to reach break-even
  - confidence: string - high|medium|low
  - assumptions: array
- margin_of_safety:
  - units_above_break_even: number - buffer if targets met
  - pct_above_break_even: number - percentage buffer
  - interpretation: string - what margin of safety means
- assumptions_used:
  - conversion_rate: number
  - revenue_per_conversion: number
  - contribution_margin_pct: number
  - fixed_costs: number
  - variable_cost_per_unit: number
- sensitivity_analysis:
  - if_conv_rate_drops_10pct:
    - new_break_even_units: number
    - pct_increase: number
  - if_conv_rate_drops_25pct:
    - new_break_even_units: number
    - pct_increase: number
  - if_revenue_drops_10pct:
    - new_break_even_units: number
    - pct_increase: number
  - if_costs_increase_10pct:
    - new_break_even_units: number
    - pct_increase: number
- critical_assumptions: array
  - assumption: string - assumption name
  - current_value: number or string
  - break_even_sensitivity: string - high|medium|low
  - monitoring_recommendation: string - how to track
- risk_assessment:
  - overall_risk: string - low|moderate|high
  - key_risks: array
  - mitigation_suggestions: array
- recommendation: string - overall assessment and guidance

INPUT PARAMETERS
----------------

- conv_rate: Text input
- fixed_costs: Text input
- investment: Text input
- margin: Text input
- revenue_per_conversion: Text input
- timeframe: Text input
- variable_costs: Text input
