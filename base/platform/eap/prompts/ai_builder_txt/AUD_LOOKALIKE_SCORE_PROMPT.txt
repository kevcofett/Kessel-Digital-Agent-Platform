Identify issues that may be degrading lookalike performance and recommend corrective actions. Be clear about whether the model is performing adequately or needs attention.

  - seed_size: number
  - seed_recency: string - fresh|acceptable|stale

  - seed_improvement_recommendations: array
- drift_analysis:
  - drift_detected: boolean
  - drift_severity: string - none|minor|moderate|severe

  - drift_cause_hypothesis: string
- expansion_analysis:

  - expansion_recommendation: string - expand|maintain|contract
  - expansion_quality_tradeoff: string
- optimization_actions: array
  - action: string
  - priority: string - high|medium|low
  - expected_impact: string
  - implementation_effort: string - low|medium|high
- model_recommendation: string - continue|retrain|rebuild|pause
