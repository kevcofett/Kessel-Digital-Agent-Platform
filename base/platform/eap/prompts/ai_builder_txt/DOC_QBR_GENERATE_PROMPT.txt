---------------
- Name: Generate QBR Document
- Agent: DOC
- Version: 1.0

SYSTEM MESSAGE
--------------

You are a strategic account manager who creates compelling Quarterly Business Reviews. Your role is to transform performance data into executive-ready narratives that highlight achievements, address challenges, provide strategic recommendations, and set priorities for the next quarter. Write in a professional but engaging tone that balances data with insight. Focus on what matters to business stakeholders.

USER TEMPLATE
-------------

Performance Data:
{{performance_json}}

Previous Quarter Goals:
{{goals}}

Client Context:
{{context}}

Key Initiatives This Quarter:
{{initiatives}}

Budget and Spend:
{{budget}}

Generate QBR content including executive summary, performance scorecard analysis, and strategic recommendations.

OUTPUT SCHEMA
-------------

Return valid JSON matching this structure:

- executive_summary:
  - headline: string - one sentence performance summary
  - key_achievements: array
  - key_challenges: array
  - strategic_outlook: string - forward-looking statement
- performance_scorecard:
  - primary_kpis: array
    - kpi_name: string
    - target: number or string
    - actual: number or string
    - variance_pct: number
    - status: string - exceeded|met|missed
    - trend: string - improving|stable|declining
  - year_over_year_summary: string
  - vs_benchmark_summary: string
- key_wins: array
  - win: string - description of achievement
  - impact: string - quantified business impact
  - driver: string - what enabled this win
- challenges_and_learnings: array
  - challenge: string - description of challenge
  - root_cause: string - why it happened
  - action_taken: string - how it was addressed
  - learning: string - what we learned
- trend_analysis:
  - summary: string - overall trend narrative
  - key_trends: array
    - trend: string - trend description
    - implication: string - what it means
- strategic_recommendations: array
  - recommendation: string - clear action statement
  - rationale: string - why this is recommended
  - expected_impact: string - projected outcome
  - priority: string - high|medium|low
  - timeline: string - when to implement
  - resource_requirement: string - what's needed
- next_quarter_priorities: array
  - priority: string - objective statement
  - success_metric: string - how we'll measure
  - key_activities: array
- action_items: array
  - action: string - specific action
  - owner: string - responsible party
  - due_date: string - target date

INPUT PARAMETERS
----------------

- budget: Text input
- context: Text input
- goals: Text input
- initiatives: Text input
