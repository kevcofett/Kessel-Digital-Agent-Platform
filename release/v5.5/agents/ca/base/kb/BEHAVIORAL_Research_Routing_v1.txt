Research Routing Guide V12

Topic-to-Document Mapping for Deep Research

Purpose

This guide directs the Consulting Agent to the correct Knowledge Base documents based on the topic detected in user requests. Use this guide during Step 4 Deep Research phase.

Always Search These Documents

For every analysis, regardless of topic:

Framework_Library_System_V12

- Retrieve selected framework methodology and step-by-step application guidance

- Extract: Framework components, when to use, application process, output templates

Industry_Expertise_Module_V12

- Retrieve industry-specific context for detected vertical

- Extract: Industry characteristics, key challenges, MarTech priorities, framework adaptations

- Industries: Financial Services, Retail, CPG, QSR, Travel, Healthcare, Technology

Topic-Based Document Routing

MEDIA PLANNING TOPICS
When the topic involves media planning, advertising, campaigns, channel mix, or budget, search the following documents. Consult Benchmarks_Media_KPIs_V12 to extract CPM, CTR, and conversion benchmarks. Consult Benchmarks_Inventory_Pricing_V12 to extract pricing by format and buying method. Consult the relevant DSP_Reference documents to extract platform capabilities and requirements.

RETAIL MEDIA TOPICS
When the topic involves RMNs, shopper marketing, or retailer ads, search the following documents. Consult RMN_Reference_Grocery_V12 to extract Amazon, Walmart, and Kroger capabilities. Consult RMN_Reference_Mass_Specialty_V12 to extract Target, Best Buy, and specialty retailer information. Consult Benchmarks_Media_KPIs_V12 to extract pricing and technical requirements.

MARTECH AND CDP TOPICS
When the topic involves customer data platforms or marketing technology, search the following documents. Consult MarTech_Reference_CDP_V12 to extract Segment, Tealium, and mParticle capabilities. Consult Identity_Resolution_Reference_V12 to extract LiveRamp and TransUnion match rates. Consult Clean_Room_Reference_Core_V12 to extract clean room providers and use cases.

DATA STRATEGY TOPICS
When the topic involves identity, privacy, or first-party data, search the following documents. Consult Identity_Resolution_Reference_V12 to extract match rates and technical requirements. Consult Clean_Room_Reference_Core_V12 to extract privacy-safe collaboration options. Consult Contextual_Targeting_Reference_V12 to extract cookieless targeting approaches.

AUDIENCE AND DEMOGRAPHICS TOPICS
When the topic involves targeting, market sizing, or demographics, search the following documents. Consult Census_Demographics_DMA_V12 to extract top 50 DMAs with TV households. Consult Census_Demographics_Regional_V12 to extract median income and market characteristics. Consult Contextual_Targeting_Reference_V12 to extract audience taxonomy and targeting information.

STRATEGIC ANALYSIS TOPICS
When the topic involves planning, transformation, or complex analysis, search the following documents. Consult Enterprise_Consulting_Tools_V12 to extract transformation roadmaps and scenario planning guidance. Consult Consulting_Tools_Module_V12 to extract ROI calculation and prioritization tools. Consult Advanced_Analytics_Module_V12 to extract measurement frameworks and CLV analysis.

Web Search Content Leakage Protection

Before executing any web search, sanitize the query to prevent leaking confidential information.

Never Include in Web Searches

- Client names or company names from conversation

- Budget figures mentioned by user

- Project names or internal code names

- Any text copied from internal documents

- Confidential strategy details

Always Use in Web Searches

- Generic industry terms (e.g., 'retail media market size 2024')

- Public company names only for publicly available information

- Standard terminology (e.g., 'CDP vendor comparison')

- Category descriptors (e.g., 'grocery retail media networks')

- Date qualifiers (e.g., 'Q4 2024 trends')

Example Sanitization

User mentions: 'Acme Corp evaluating CDPs with $2M budget'

Correct web search: 'enterprise CDP market comparison 2024'

Incorrect web search: 'Acme Corp CDP $2M budget'
