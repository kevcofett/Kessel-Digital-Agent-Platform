GlossaryV50 V7.0

Term Definitions and Concepts

Version 7.0 - December 2025

Document Metadata

Document: GlossaryV6_1.docx version: 7.0 Last Updated: 2025-12-04 Purpose: Define terminology used throughout V7.0 documentation Owner: System Administrator Status: Active

A

Agent Instructions

The configuration content loaded into CoPilots agent instructions field. Limited to 8,000 characters. Contains core behavioral rules.

Anti-Pattern

A common behavior the system should avoid. V7.0 defines eight specific anti-patterns.

Approval Gate

A checkpoint requiring explicit user approval before proceeding. Step 9 is a mandatory approval gate.

B

Banned Phrase

A phrase the system must never use in outputs. V7.0 defines seven banned phrases related to system limit excuses.

Blocking Gate

A workflow step that halts execution until a condition is met. Step 9 checkpoint is a blocking gate.

C

Checkpoint

A pause in workflow for user review and approval. Step 9 presents the optimization checkpoint.

Complexity Tier

Classification of request complexity. Four tiers: Simple (7-10 points), Moderate (11-15), Complex (16-18), Enterprise (19-21).

CoPilot

Microsofts AI assistant platform where Aragorn AI is deployed.

D

Deliverable

The final output produced in Step 5. Format and length based on complexity tier.

E

Engine

Short for Prompt Execution Engine. The component that executes the 12-step optimization workflow.

Enhancement Request

A user submission proposing system improvements. Tracked in Enhancement Request Log.

F

Failure Mode

A specific way the system can fail to meet expectations. V7.0 addresses four failure modes.

Framework

A structured analytical methodology. 32 frameworks available in the Framework Library.

Framework Library

The collection of 32 consulting frameworks organized in 5 packs.

Framework Pack

A group of related frameworks that load together. Five packs available.

G

Gap

A discrepancy between expected and actual system behavior. Tracked by Gap Correction System.

K

Knowledge Base

The collection of documents providing system knowledge. Stored in SharePoint.

M

----------------------------------------
MECE
----------------------------------------

Mutually Exclusive, Collectively Exhaustive. A structuring principle ensuring no overlaps and no gaps.

Mode

Execution mode controlling workflow detail. Three modes: Quick, Full, Expert.

O

Optimization

The process of enhancing a user prompt through the 12-step workflow.

Output

The deliverable generated in Step 5.

P

Pack

Short for Framework Pack.

Page Minimum

Minimum page count required for deliverable. Varies by complexity tier.

Phased Delivery

Delivering output in multiple phases with user confirmation. Required for Complex and Enterprise tiers.

Profile

The agent expertise profile applied to analysis. Default is Aragorn AI Profile V7.0.

Prose

Full narrative paragraph format. Required for deliverables instead of bullet points.

R

Role

User access level. Five roles: Novice, Standard, Advanced, Expert, Administrator.

S

Session

A single optimization interaction from start to completion.

SharePoint

Microsofts document management platform where knowledge base is stored.

Step

One of ten stages in the optimization workflow.

T

Tier

Short for Complexity Tier.

Token

Unit of text measurement for AI models. Token budget limits total text processed.

Token Budget

Maximum tokens allowed for a session. Varies by role and session type.

Trigger Word

A phrase that might cause unintended system behavior. V7.0 includes trigger word protection.

V

----------------------------------------
V7.0
----------------------------------------

Version 5.0 of Aragorn AI. Current production version.

Validation

Verification that conditions are met before proceeding.

W

Workflow

The 12-step optimization process.
