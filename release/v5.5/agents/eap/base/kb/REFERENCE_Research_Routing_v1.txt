OVERVIEW

This document provides guidance on routing research queries to appropriate knowledge sources. Use this to determine which KB files or external sources to consult for specific topics.

EXTERNAL RESEARCH TRIGGERS

Web Research Required When:
- Current market data needed (last 6 months)
- Competitor-specific information
- Stock prices or financial results
- Regulatory updates
- News events
- Technology product releases
- Industry reports and studies

Web Search Protection Rules:
- NEVER include client names in web searches
- NEVER include budget figures
- NEVER include internal project names
- USE only generic industry terms
- SEARCH for publicly available information only

CONFIDENCE LEVEL MAPPING

By Source Type

Dataverse (structured data) has HIGH confidence. This represents internal authoritative data.

Platform KB files have HIGH confidence. These are curated consulting knowledge resources.

Client documents have HIGH confidence. These are client-provided context materials.

Industry reports (recent) have MEDIUM-HIGH confidence. These are published research from reputable sources.

Web search (current) has MEDIUM confidence. Results should be verified across multiple sources.

Web search (older) has MEDIUM-LOW confidence. Information may be outdated and should be used cautiously.

General web content has LOW confidence. This requires validation before use.

SEARCH STRATEGIES

For Framework Questions:
- Search FRAMEWORK_Library first
- Check if industry context needed from INDUSTRY_Vertical
- Add benchmarks from BENCHMARK_Industry if quantitative

For Industry Questions:
- Start with INDUSTRY_Vertical for context
- Add BENCHMARK_Industry for metrics
- Use FRAMEWORK_Library for analysis structure

For Methodology Questions:
- Check TOOLS_Consulting first
- Reference FRAMEWORK_Library for specific frameworks
- Add industry context if sector-specific

For Benchmarking Questions:
- Search BENCHMARK_Industry for metrics
- Add INDUSTRY_Vertical for context
- Note confidence levels and data sources

ESCALATION PATHS

If KB Search Yields No Results:
- Broaden search terms
- Check related categories
- Use web search for external sources
- Acknowledge gap and ask client for context

If Information Conflicts:
- Prefer more recent sources
- Prefer authoritative sources (industry reports over blogs)
- Note the conflict and present both views
- Recommend validation if critical to decision

If Information Is Outdated:
- Note the date of information
- Search for more recent data
- Adjust for known trends
- Flag uncertainty in confidence level
