================================================================================
AVAILABLE INTEGRATIONS REGISTRY
Enterprise AI Platform v1.0
================================================================================

PURPOSE
-------
This registry lists all external integrations that can be connected to the
platform. Integrations provide access to external data, services, and APIs
that enhance agent capabilities.

INTEGRATION STATUS
------------------
Each integration has a status indicating its connection state:

- AVAILABLE: Integration is defined but not yet connected
- CONNECTED: Credentials configured, ready to use
- ACTIVE: Enabled and in use
- DEGRADED: Experiencing issues, limited functionality
- UNAVAILABLE: Temporarily offline

NIELSEN AUDIENCE MEASUREMENT
----------------------------
Code: nielsen
Type: Measurement
Agent Domains: MPA
Status: AVAILABLE
Description: Audience measurement, demographic data, reach and frequency
             analysis for media planning optimization.

Operations Available:
- get-audience-data: Retrieve audience segment demographics and size
- get-reach-frequency: Calculate reach/frequency for media schedules
- get-benchmarks: Industry benchmark data by vertical

Data Provided:
- Audience segment sizes and demographics
- Viewing behavior patterns

- Brand lift studies

To Connect: Contact your Nielsen representative for API credentials.

================================================================================
REQUESTING NEW INTEGRATIONS
================================================================================

To request a new integration:

- Identify the service you need to connect
- Confirm API access is available
- Document:
   - Service name and provider
   - What data/operations you need
   - Authentication type required
   - Expected usage volume
- Submit request to platform administrators
- Await credential configuration and testing
- Integration will appear as CONNECTED when ready

================================================================================
INTEGRATION HEALTH
================================================================================

Current health status is checked every 5 minutes. If an integration shows
degraded or unavailable status:

- The agent will acknowledge the limitation
- Cached data may be used if available
- Recommendations will be based on available data
- You can request a retry when status returns to healthy
