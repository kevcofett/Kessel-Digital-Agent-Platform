ADIS QUICK REFERENCE CARD

PURPOSE

Fast reference for Audience Data Intelligence System capabilities and triggers.

FILE UPLOAD

SUPPORTED FORMATS
- Excel XLSX XLS up to 250MB
- CSV comma or semicolon delimited
- Word DOCX with tables
- PDF with extractable tables
- PowerPoint PPTX with data tables

MINIMUM REQUIREMENTS
- Customer identifier column
- At least one behavioral or transactional column
- Minimum 50 records for most analyses

- I have customer data to upload
- Here is my transaction file
- Can you analyze this customer list
- I want to upload my CRM export

Analysis Triggers:
- Run RFM analysis
- Calculate customer lifetime value
- Show me cohort retention
- Cluster my customers
- Segment my customers

Audience Triggers:
- Create an audience of champions
- Target my best customers
- Build segments for activation
- Export audience to platform

Optimization Triggers:
- Optimize budget for my audiences
- How should I allocate spend by segment
- Run AMMO optimization

ANALYSIS MODELS

RFM SEGMENTATION
- Required: customer_key transaction_date monetary_value
- Minimum: 100 customers 500 transactions
- Output: 11 named segments with R F M scores

DECILE ANALYSIS
- Required: customer_key value_metric
- Minimum: 50 customers
- Output: 10 value tiers with Pareto ratio

COHORT ANALYSIS
- Required: customer_key transaction_date
- Minimum: 200 customers across 3 periods
- Output: Retention matrix by cohort

DETERMINISTIC CLV
- Required: customer_key transaction_date monetary_value
- Minimum: 100 customers with 90 day history
- Output: Projected lifetime value per customer

PROBABILISTIC CLV
- Required: customer_key transaction_date monetary_value
- Minimum: 500 customers with 180 day history
- Output: Statistical CLV with confidence intervals

K-MEANS CLUSTERING
- Required: At least 2 numeric features
- Minimum: 500 records
- Output: Behavioral clusters with centroids

RFM SEGMENTS

Champions - Best customers highest engagement highest value
Loyal Customers - Consistent purchasers reliable value
Potential Loyalists - Recent customers showing promise
New Customers - Recent first-time purchasers
Promising - Recent with high value but low frequency
Needs Attention - Good customers showing early decline
About To Sleep - Below average declining engagement
At Risk - Previously valuable now concerning
Cannot Lose Them - High value at serious churn risk
Hibernating - Dormant with historical value
Lost - Inactive minimal current value

CHANNEL AFFINITY BY SEGMENT

Champions:
- High affinity: Email 1.4 Direct Mail 1.5 CTV 1.3
- Moderate: Paid Social 1.2 Programmatic 1.1
- Low affinity: OOH 0.7

At Risk:
- High affinity: Email 1.5 Direct Mail 1.4 Programmatic 1.3
- Moderate: Paid Social 1.1
- Low affinity: Linear TV 0.7 OOH 0.6

Lost:
- All channels below 1.0 - suppress from active targeting

AUDIENCE RULE OPERATORS

EQUALS exact match
NOT_EQUALS exclusion
GREATER_THAN numeric threshold
GREATER_THAN_OR_EQUAL inclusive threshold
LESS_THAN numeric ceiling
LESS_THAN_OR_EQUAL inclusive ceiling
IN list membership
NOT_IN list exclusion
BETWEEN range filter
CONTAINS substring match
STARTS_WITH prefix match
IS_NULL null check
IS_NOT_NULL presence check

AMMO OPTIMIZATION

INPUTS REQUIRED
- Total budget amount
- Audience definitions with member counts and values
- Channels to consider
- Optional constraints min max percentages
- Optional objective ROI REVENUE CONVERSIONS

OUTPUTS PROVIDED
- Allocation percentages by channel
- Budget amounts by channel
- Expected impressions conversions revenue
- Expected ROI
- Comparison to current allocation if provided

EXPORT FORMATS

CSV - Standard comma-separated for general use
GOOGLE_ADS - Customer Match format with hashed identifiers
META - Custom Audiences format for Facebook Instagram
TTD - Trade Desk segment format

PII HANDLING OPTIONS
- Include raw identifiers for internal use
- Exclude all PII for external platforms
- Hash with SHA256 for secure matching
- Hash with MD5 for legacy platform compatibility
