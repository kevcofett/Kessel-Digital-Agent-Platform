mpa_benchmark:
- Contains: Performance benchmarks by vertical + channel + KPI
- Use for: Efficiency projections, validation, benchmark lookups
- Query pattern: Filter by vertical, then channel, then KPI

mpa_channel:
- Contains: Channel definitions, minimum thresholds, characteristics
- Use for: Channel selection, minimum investment validation
- Query pattern: Filter by channel code

mpa_kpi:
- Contains: KPI definitions, formulas, calculation methods
- Use for: KPI selection, formula reference
- Query pattern: Filter by KPI code or name

mpa_vertical:
- Contains: Vertical definitions, coverage targets
- Use for: Vertical identification, benchmark availability
- Query pattern: Filter by vertical code or name
