1.1 PURPOSE

This document provides the structural framework for platform-specific audience taxonomies. Activation-ready audience specifications require platform-specific taxonomy codes.

1.2 CORE PRINCIPLE

The agent must web search for current codes rather than relying solely on this reference, as platforms update taxonomies quarterly. This document provides structure and search patterns, not a complete code database.
