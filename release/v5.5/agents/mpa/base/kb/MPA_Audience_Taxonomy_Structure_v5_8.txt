WEB SEARCH EXECUTION RULES

Rule 1 - Search before assuming
Never assume a taxonomy code is correct. Always verify with web search.

Rule 2 - Cite every data point
Population figures and taxonomy codes must include source attribution.

Rule 3 - Prefer official sources
Platform help docs and national statistical offices are authoritative.

Rule 4 - Date the data
Include year or date range for census data. Note when taxonomy was verified.

Rule 5 - Flag uncertainty
If search does not return definitive results, note this in the specification.

OUTPUT QUALITY CHECKLIST

Before completing audience specification:
- All IAB codes follow [IAB##-##] format
- All Google audiences follow [/Type/Category/Subcategory] format
- All Meta interests verified as currently available
- All LinkedIn attributes match platform options
- All population figures cite census source and year
- All audience size calculations show methodology
- Total signal count meets campaign type threshold
