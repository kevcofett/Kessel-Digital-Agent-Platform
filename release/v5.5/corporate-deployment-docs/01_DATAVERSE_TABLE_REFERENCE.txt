MPA V5.5 CORPORATE DEPLOYMENT

============================================================
OVERVIEW
============================================================

This document provides complete specifications for all 8 MPA Dataverse tables required for deployment. Tables must be created in the order listed due to lookup dependencies.

CREATION ORDER
1 mpa_vertical - No dependencies
2 mpa_channel - No dependencies
3 mpa_kpi - No dependencies
4 mpa_benchmark - No dependencies (uses text references)
5 mpa_errorlog - No dependencies
6 mpa_mediaplan - No dependencies
7 mpa_planversion - Depends on mpa_mediaplan
8 mpa_plandata - Depends on mpa_mediaplan and mpa_planversion

============================================================
PREREQUISITE - SOLUTION AND PUBLISHER
============================================================

STEP 1 - CREATE PUBLISHER

Navigation
1 Open make.powerapps.com
2 Select your environment
3 Click Solutions
4 Click New solution
5 Click New publisher at bottom

Publisher Settings
- Display name - Media Planning Agent
- Name - mediaplanning
- Prefix - mpa
- Choice value prefix - 10000

Click Save

STEP 2 - CREATE SOLUTION

Solution Settings
- Display name - Media Planning Agent v5.5
- Name - MediaPlanningAgent
- Publisher - Media Planning Agent (mpa)
- Version - 5.5.0.0

Click Create

IMPORTANT - All tables MUST be created from inside this solution
