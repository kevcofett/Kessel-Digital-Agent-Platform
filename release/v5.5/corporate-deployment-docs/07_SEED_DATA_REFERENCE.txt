IMPORT METHODS
============================================================

METHOD 1 - POWER APPS DATA IMPORT

Steps
1 Open make.powerapps.com
2 Select environment
3 Open table
4 Click Import then Import data
5 Select Import from CSV
6 Choose file
7 Map columns
8 Click Import

Column Mapping
- Verify each CSV column maps to correct table column
- Handle any name differences
- Set data types correctly

METHOD 2 - PYTHON SCRIPT

Prerequisites
- Python 3.9 or higher
- MSAL library installed
- Network access to Dataverse

Script Location
release/v5.5/scripts/seed_data_import.py

Usage
python seed_data_import.py --table all
python seed_data_import.py --table vertical
python seed_data_import.py --table channel
python seed_data_import.py --table kpi
python seed_data_import.py --table benchmark
python seed_data_import.py --dry-run

Authentication
- Script uses device code flow
- User authenticates via browser
- Token cached for subsequent runs

METHOD 3 - PAC CLI

Prerequisites
- Power Platform CLI installed
- Authenticated to environment

Commands
pac auth create --url https://environment.crm.dynamics.com
pac data import --file mpa_vertical_seed.csv --table mpa_vertical
pac data import --file mpa_channel_seed.csv --table mpa_channel
pac data import --file mpa_kpi_seed.csv --table mpa_kpi
pac data import --file mpa_benchmark_seed.csv --table mpa_benchmark
