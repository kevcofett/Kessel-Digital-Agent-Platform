CLUSTERING EVALUATION

Assessing cluster quality.

INTERNAL METRICS

No ground truth evaluation.

Silhouette score
- Cohesion versus separation
- Range negative one to one
- Higher is better
- Per-cluster analysis

Davies-Bouldin index
- Cluster separation ratio
- Lower is better
- Average across clusters
- Sensitive to outliers

Calinski-Harabasz index
- Variance ratio
- Higher is better
- Well-separated clusters
- Computationally efficient

EXTERNAL METRICS

Ground truth comparison.

Adjusted Rand Index
- Pair agreement
- Chance corrected
- Range negative to one
- Perfect at one

Normalized Mutual Information
- Information theoretic
- Range zero to one
- Higher is better
- Handles different K
