WHAT IS V3.6?

V3.6 UNIFIED combines two V3.5 branches:
- Branch A: Three-tier enforcement (fixes 20-30% execution failures)
- Branch B: 29 enhancement modules (rich capabilities)
- Result: Reliable core + powerful features

KEY INNOVATION: Enforcement tiers ensure 95%+ checkpoint execution while enhancement modules provide sophisticated capabilities on-demand.

------------------------------------------------------------

CORE ARCHITECTURE

THREE-TIER ENFORCEMENT

TIER 1 - MANDATORY (5 checkpoints, always execute, no override)
1. Prompt Assessment
2. Complexity Classification
3. Token Estimation
4. Framework Recommendation
5. Pre-Flight Validation

TIER 2 - CONDITIONAL (3 checkpoints, enforce when triggered)
1. Model Selection (complexity 7+)
2. Gap Correction (critical gaps)
3. Profile Confirmation (custom profile)

TIER 3 - FLEXIBLE (29 modules, user-controlled)
S-Stability, E-Efficiency, Q-Quality, U-UX, P3-Power

DUAL WORKFLOW

NEW TASK: STEP 0 (6 steps) + MAIN (10 steps)
ONGOING: MAIN (10 steps) only

Mode affects verbosity only, not checkpoint execution.

U-UX (9): Undo, help, quick mode, indicators, scope alert, pre-check, transparency, session mgmt, status
P3-POWER (5): Batch processing, templates, checkpoints, consistency guard, confidence scoring
