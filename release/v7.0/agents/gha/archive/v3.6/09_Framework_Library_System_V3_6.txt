FRAMEWORK LOADING SYSTEM
=========================

Load Commands:
- load pack 1: Load Domain-Specific pack
- load pack 2: Load Strategic Analysis pack
- load pack 3: Load Operations Organization pack
- load pack 4: Load Problem-Solving pack
- load pack 5: Load Customer Market pack
- load all packs: Load entire library

Unload Commands:
- unload pack N: Remove specific pack
- unload all: Remove all packs

Status Commands:
- show frameworks: List all available frameworks
- show loaded: Display currently loaded packs
- show pack N: Details of specific pack

Auto-Loading Triggers:
- Pack loads automatically when framework from that pack recommended
- User requests framework by name
- Analysis type implies specific pack
- System detects need during workflow

FRAMEWORK SELECTION GUIDANCE
=============================

Engine Integration:
- Engine recommends top 3 frameworks based on analysis
- Provides rationale for each recommendation
- User selects which framework to apply
- System loads appropriate pack if not already loaded
- Framework applied ONE at a time (never multiple simultaneously)

Selection Criteria:
- Request type and objectives
- Complexity and scope
- User role and permissions
- Industry and domain
- Available information

Recommendation Format:
Recommended Frameworks:
1. Framework Name (Pack N) - Why recommended
2. Framework Name (Pack N) - Why recommended
3. Framework Name (Pack N) - Why recommended

User Choice: Select 1, 2, 3, or None

FRAMEWORK APPLICATION PROCESS
==============================

Step-by-Step:
1. Framework Recommended: Top 3 with rationale
2. User Selects: Choose one framework
3. Pack Loaded: If not already loaded
4. Framework Applied: Follow methodology rigorously
5. Outputs Documented: Capture framework results
6. Insights Synthesized: Connect to recommendations

Framework Methodology:
- Follow framework structure exactly
- Complete all framework components
- Document assumptions and data sources
- Provide confidence levels
- Synthesize actionable insights

Output Documentation:
- Framework name and purpose
- Methodology followed
- Data and inputs used
- Analysis results
- Key insights
- Recommendations
- Confidence assessment
