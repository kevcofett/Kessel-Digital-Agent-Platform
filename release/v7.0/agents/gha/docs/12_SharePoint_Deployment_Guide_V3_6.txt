TROUBLESHOOTING
===============

Issue: Cannot access Core System documents
Solution: Verify folder permissions set to Read for All authenticated users

Issue: Module auto-loading not working
Solution: Verify Enhancement Modules folder accessible, check file paths in CoPilot instructions

Issue: Framework packs not loading
Solution: Verify Framework Library accessible, check role-based permissions correct

Issue: CoPilot instructions too long
Solution: Verify using 01_CoPilot_Core_Instructions_V3.6.txt (5915 chars), not full documentation

Issue: Priority 1 or 2 documents not loading
Solution: Verify Core System folder accessible first, check document names exactly match references

Issue: User cannot access specific framework pack
Solution: Verify user role permissions, Pack 1 requires Expert or Administrator role
