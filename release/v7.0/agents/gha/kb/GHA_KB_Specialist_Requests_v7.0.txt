GHA KNOWLEDGE BASE - SPECIALIST REQUEST PATTERNS

OVERVIEW

GHA is a Growth Strategy Orchestrator that coordinates with specialist agents via ORC to quantify and validate growth plans. This knowledge base defines when GHA should request specialist assistance and how to structure those requests.

REQUESTING SPECIALISTS VIA ORC

GHA does not call specialists directly. Instead, GHA sends structured requests to ORC, which routes to the appropriate specialist and returns the result. This maintains ORC as the central orchestration hub while allowing GHA to leverage specialist capabilities.

Request Format:
- Clearly state the specialist needed and the specific task
- Provide relevant growth context from the current session
- Specify the output format needed
- Request response under 300 words to maintain conversation flow

ANL AGENT REQUESTS

Request Analytics Agent assistance when:
- Growth strategy needs ROI or ROAS projections
- LTV and CAC calculations required for growth segments
- Scenario modeling for growth initiative investments
- Statistical validation for experiment results
- Financial projections for growth program funding
- Diminishing returns analysis for scaling decisions

Keywords indicating ANL need: ROI, ROAS, LTV, CAC, projection, scenario, forecast, statistical significance, break-even, payback period

Example request to ORC: I need ANL to project the expected ROI for this referral program. Budget is 50K over 6 months, expected referral rate is 12 percent, and average customer LTV is 850 dollars. Please calculate conservative, base, and optimistic scenarios.

AUD AGENT REQUESTS

Request Audience Agent assistance when:
- Segmentation needed for growth targeting
- LTV tier analysis required for prioritization
- Persona development for growth tactics
- Lookalike strategy for acquisition scaling
- Behavioral segmentation for lifecycle stages
- RFM analysis for retention targeting

Keywords indicating AUD need: segment, persona, LTV tiers, lookalike, behavioral, targeting, RFM, cohort, propensity

Example request to ORC: I need AUD to segment our users by lifecycle stage. Please identify acquisition, activation, retention, and at-risk segments with estimated sizes and LTV per segment.

CHA AGENT REQUESTS

Request Channel Agent assistance when:
- Growth tactics need channel recommendations
- Budget allocation for growth initiatives
- Platform-specific growth tactics with benchmarks
- Channel mix optimization for growth goals
- Frequency and reach estimates for growth campaigns

Keywords indicating CHA need: channel, allocation, platform, benchmark, media mix, CPM, CPC, reach, frequency

Example request to ORC: I need CHA to recommend channel allocation for a 200K acquisition campaign targeting fintech early adopters. Focus on channels with strong performance for app install and account activation goals.

DOC AGENT REQUESTS

Request Document Agent assistance when:
- Growth strategy needs formal documentation
- Growth plan export required for stakeholders
- Executive summary for growth initiative
- Presentation for growth recommendation
- Business case document for growth investment

Keywords indicating DOC need: document, export, summary, presentation, brief, business case, deck

Example request to ORC: I need DOC to generate an executive summary of this growth strategy. Include the North Star metric, priority lifecycle stages, key tactics, and expected outcomes. Format for C-level presentation.

PRF AGENT REQUESTS

Request Performance Agent assistance when:
- Growth campaign results need analysis
- Optimization recommendations for running experiments
- Anomaly detection in growth metrics
- Post-mortem analysis for growth initiatives
- Attribution analysis for growth channels

Keywords indicating PRF need: performance, results, metrics, optimization, anomaly, post-mortem, attribution

Example request to ORC: I need PRF to analyze the results of our referral program test. Compare control versus test groups on activation rate, retention at day 30, and LTV. Flag any anomalies or statistical concerns.

MKT AGENT REQUESTS

Request Marketing Strategy Agent assistance when:
- Growth tactics need campaign strategy support
- Creative brief needed for growth campaign
- Messaging framework for growth positioning
- Competitive positioning for growth narrative

Keywords indicating MKT need: campaign strategy, creative brief, messaging, positioning, competitive

Example request to ORC: I need MKT to develop a messaging framework for our referral program. Position the value proposition for both referrers and referred users. Focus on fintech trust and reward clarity.

CST AGENT REQUESTS

Request Consulting Strategy Agent assistance when:
- Growth initiative needs strategic framework
- Prioritization help for growth experiments
- Business model analysis for growth strategy
- Maturity assessment for growth capabilities

Keywords indicating CST need: framework, prioritization, strategic analysis, maturity, assessment

Example request to ORC: I need CST to help prioritize these 8 growth experiments using RICE scoring. Provide the ranked list with scores and rationale for the top 3.

MULTI-AGENT SEQUENCES

GHA often needs multiple specialists in sequence. Always wait for each specialist response before requesting the next.

Growth Strategy Development Sequence:
- GHA defines growth framework and North Star metric
- Request AUD for target segment definitions
- Request ANL for LTV and CAC projections per segment
- Request CHA for channel recommendations per segment
- GHA synthesizes into complete growth plan
- Request DOC for formal documentation

Lifecycle Optimization Sequence:
- GHA identifies priority lifecycle stage
- Request AUD for stage-specific segment analysis
- Request ANL for stage-specific projections
- GHA develops tactics and experiments
- Request DOC for experiment brief

Growth to Media Plan Handoff:
- GHA completes growth strategy with priorities
- GHA requests handoff to MPA domain via ORC
- ORC routes subsequent media planning requests with growth context
- Media plan incorporates growth strategy inputs

CONTEXT PRESERVATION

When requesting specialists, always provide relevant growth context:

- Current growth stage and North Star metric
- Priority lifecycle stages identified
- Target segments and their characteristics
- Budget and timeline constraints
- Key assumptions and success criteria

This ensures specialists have the context needed to provide relevant recommendations that align with the growth strategy.

RESPONSE HANDLING

When specialist responses return via ORC:

- Integrate the insights into the growth plan
- Validate alignment with growth objectives
- Identify any conflicts or gaps
- Update growth state with specialist outputs
- Continue to next step or next specialist as needed

Never proceed to documentation until all required specialist inputs are integrated and validated.
