# GROWTH KPI DEFINITIONS, BENCHMARKS & FIXES

--------------------------------------------------
KPI: Activation Rate
Definition: % of sign-ups who complete first meaningful action
Red Flag: <40% = friction or unclear value
Benchmarks:
- Fintech: 60–75%
- SaaS: 40–60%
Fix Playbooks:
- B-1: Speedrun Onboarding
- B-6: Time-limited First Swipe Reward
- B-9: Push + SMS Combo

--------------------------------------------------
KPI: Usage Frequency
Definition: Average # of transactions per user per month
Red Flag: <2/month = forgotten card
Benchmarks:
- Credit Cards: 4–8/month
Fix Playbooks:
- C-2: Spending Streak
- C-1: Category Gamification
- D-10: Loss Aversion Push

--------------------------------------------------
KPI: Referral Rate
Definition: % of users inviting friends
Red Flag: <5% = no virality
Benchmarks:
- Fintech: 8–15%
- Neobank: 10–20%
Fix Playbooks:
- E-1: Mystery Box Referral
- E-9: Status Badge Ladder
- E-14: Screenshot Flex Contest

--------------------------------------------------
KPI: Reactivation Rate
Definition: % of dormant users re-engaged in 30 days
Red Flag: <10%
Benchmarks:
- Fintech: 15–25%
Fix Playbooks:
- G-1: Break-Up Email
- G-2: Countdown Perk
- G-3: Comeback Story
