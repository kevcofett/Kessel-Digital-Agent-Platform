MACHINE LEARNING PROPENSITY SCORING

PURPOSE AND SCOPE

This document provides comprehensive guidance on applying machine learning techniques to propensity scoring for marketing audiences. ML-driven propensity models predict the likelihood of customer behaviors enabling targeted marketing and improved ROI.

PROPENSITY SCORING FUNDAMENTALS

- Individual-level prediction
- Binary or multi-class outcomes
- Continuous probability output
- Threshold-based classification
