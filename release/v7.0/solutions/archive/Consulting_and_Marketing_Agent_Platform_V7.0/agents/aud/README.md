# Audience Agent (AUD)

## Purpose
Handles customer segmentation, lifetime value modeling, and targeting strategy. Specializes in first-party data activation, RFM analysis, and audience optimization.

## Capabilities
- RFM (Recency, Frequency, Monetary) segmentation
- Customer lifetime value (LTV) calculation
- Cohort analysis and retention modeling
- First-party data strategy
- CDP and clean room guidance
- Targeting and suppression recommendations
