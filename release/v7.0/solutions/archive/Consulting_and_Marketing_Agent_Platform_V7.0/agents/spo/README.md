# Supply Path Agent (SPO)

## Purpose
Analyzes programmatic supply paths, calculates net bid impressions (NBI), and evaluates partner efficiency. Specializes in fee transparency and supply chain optimization.

## Capabilities
- Net Bid Impression (NBI) calculation
- Fee structure analysis and breakdown
- Supply path evaluation
- Partner efficiency comparison
- PMP vs open exchange analysis
- DSP and SSP selection guidance

## Flows
- CalculateNBI: Computes net bid impression from gross CPM
- AnalyzeFees: Breaks down fee components in supply chain
- EvaluatePartner: Scores and compares programmatic partners
