SECTION 1 - DOCX EXPORT SPECIFICATIONS
================================================================================

DOCX FORMAT OVERVIEW

Microsoft Word DOCX is the primary format for editable documents. Use DOCX when clients need to review, comment, or modify content.

TECHNICAL SPECIFICATIONS

FILE FORMAT
- Format: Office Open XML (.docx)
- Compatibility: Word 2016 and later recommended
- Backwards compatible: Word 2007 and later

DOCUMENT PROPERTIES
- Title: Campaign name and document type
- Author: Kessel Digital Agent Platform
- Subject: Media planning document
- Keywords: Campaign name, client, date

GENERATION REQUIREMENTS

CONTENT POPULATION
- All placeholders replaced with actual data
- No visible template markers in output
- Tables fully populated (no empty rows unless intentional)
- Page breaks at section boundaries

STYLING APPLICATION
- Consistent heading styles applied
- Table styles uniformly formatted
- Font substitution avoided
- Color palette from template preserved

EMBEDDED ELEMENTS
- Charts: Embedded as editable objects
- Images: Embedded at print resolution (300 DPI)
- Tables: Native Word tables (not images)

OUTPUT VALIDATION
Before delivery, verify:
- File opens without errors
- All sections present and populated
- Formatting renders correctly
- Page numbers accurate
- Table of contents updated

SECTION 2 - PDF EXPORT SPECIFICATIONS
================================================================================

PDF FORMAT OVERVIEW

PDF is the format for final, locked documents. Use PDF for distribution, archival, and formal delivery.

TECHNICAL SPECIFICATIONS

FILE FORMAT
- Format: PDF/A-1b for archival (preferred)
- Alternative: Standard PDF 1.7
- Compatibility: All PDF readers

DOCUMENT PROPERTIES
- Title: Campaign name and document type
- Author: Kessel Digital Agent Platform
- Subject: Media planning document
- Created date: Generation timestamp

GENERATION REQUIREMENTS

CONVERSION FROM DOCX
- Convert from finalized DOCX
- Preserve all formatting
- Embed all fonts
- Maintain hyperlinks

PDF-SPECIFIC SETTINGS
- Bookmarks: Create from headings
- Security: No restrictions (editable by default)
- Compression: Standard (balance quality and size)
- Resolution: 300 DPI for images

ACCESSIBILITY
- Tagged PDF structure
- Reading order defined
- Alt text for images
- Document language set

OUTPUT VALIDATION
Before delivery, verify:
- File opens in standard PDF readers
- All pages render correctly
- Bookmarks navigate properly
- File size reasonable (under 10MB typical)
- Text is searchable and selectable

SECTION 3 - PPTX EXPORT SPECIFICATIONS
================================================================================

PPTX FORMAT OVERVIEW

PowerPoint PPTX is the format for presentations. Use PPTX for stakeholder meetings and visual summaries.

TECHNICAL SPECIFICATIONS

FILE FORMAT
- Format: Office Open XML (.pptx)
- Compatibility: PowerPoint 2016 and later
- Aspect ratio: 16:9 widescreen (default)

SLIDE SPECIFICATIONS
- Slide size: 13.33 x 7.5 inches (16:9)
- Alternative: 10 x 7.5 inches (4:3)
- Resolution: 96 DPI standard, 300 DPI for print

GENERATION REQUIREMENTS

SLIDE CONTENT
- One key message per slide
- Maximum 6 bullet points per slide
- Bullet text under 2 lines each
- Speaker notes populated

VISUAL ELEMENTS
- Charts: Editable PowerPoint charts
- Images: High resolution embedded
- Tables: Native PowerPoint tables
- Icons: Vector format preferred

MASTER SLIDE USAGE
- Apply template master slides
- Consistent branding throughout
- Footer with date and page number

ANIMATION AND TRANSITIONS
- Default: No animations
- Transitions: None or subtle fade
- Avoid distracting effects

OUTPUT VALIDATION
Before delivery, verify:
- All slides render correctly
- Charts are editable
- No broken links
- Consistent formatting throughout
- Presenter notes complete

SECTION 4 - XLSX EXPORT SPECIFICATIONS
================================================================================

XLSX FORMAT OVERVIEW

Excel XLSX is the format for budget details and data exports. Use XLSX when finance teams need editable data.

TECHNICAL SPECIFICATIONS

FILE FORMAT
- Format: Office Open XML (.xlsx)
- Compatibility: Excel 2016 and later
- Maximum rows: 1,048,576 (Excel limit)

WORKBOOK STRUCTURE
- Tab 1: Summary (high-level allocation)
- Tab 2: Channel Detail (line items)
- Tab 3: Monthly Pacing (time-based view)
- Tab 4: Assumptions (inputs and benchmarks)

GENERATION REQUIREMENTS

DATA FORMATTING
- Currency: Formatted with dollar sign and commas
- Percentages: Formatted as percent with one decimal
- Dates: Formatted consistently (MM/DD/YYYY)
- Numbers: Right-aligned with thousand separators

CELL FORMATTING
- Headers: Bold with background color
- Data: Regular with alternating row colors
- Totals: Bold with border above
- Formulas: Use formulas where appropriate

WORKBOOK FEATURES
- Freeze panes: Header row frozen
- Column widths: Auto-fit to content
- Print area: Defined for each sheet
- Named ranges: For key totals

OUTPUT VALIDATION
Before delivery, verify:
- All formulas calculate correctly
- No circular references
- Data sums match totals
- Formatting consistent across sheets
- File opens without errors

SECTION 5 - DELIVERY AND STORAGE
================================================================================

FILE DELIVERY

DELIVERY METHODS
- SharePoint link: Preferred for collaboration
- OneDrive link: Alternative cloud delivery
- Email attachment: For small files under 10MB
- Download link: Direct file access

LINK SETTINGS
- Access: Specific people or anyone with link
- Permissions: View only or edit as appropriate
- Expiration: Set if time-limited access needed

FILE STORAGE

STORAGE LOCATION
- Primary: SharePoint document library
- Path: /[Client]/[Campaign]/Documents/
- Naming: Per naming convention

RETENTION
- Active campaigns: Retain during campaign
- Completed campaigns: Archive after 90 days
- Retention period: Per client agreement

VERSION MANAGEMENT
- Keep previous versions accessible
- Mark current version clearly
- Archive superseded versions
